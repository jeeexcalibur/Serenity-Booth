## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/jeeexcalibur/Serenity-Booth.git
   cd Serenity-Booth
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. Open `http://localhost:5173` in your browser.

## Deployment

This project is ready to be deployed on platforms like [Vercel](https://vercel.com) or [Netlify](https://netlify.com).

1. Fork this repository.
2. Import the project into Vercel/Netlify.
3. The build command is `npm run build` and output directory is `dist`.
